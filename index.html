Atividades

  1. Seleção e preparação das variáveis
    1. Diagrama de disperção dos valores (visualizar a linearidade)
    2. Matriz de correlação: identificar se as variáveis independentes são independentes entre si -> análise de multicolinearidade
    3. Linearizar as relações entre Xi e Y (probit ou logit), dado os dados nao lineares
  1. Escolha e ajuste do modelo de regressão
    1. Executar a saída para todos os previsores (Xi) e examinar a saída, observando quais contribuem substancialmente. *análise stepwise
  1. Diagnóstico de adequação do modelo
    1. R^2: Quanto, em porcentagem, as variáveis independentes explicam Y Teste F, Teste t para coef. (Teste de hipóteses, onde beta i = 0 é a hip. nula -> não influencia o modelo)
    2. Multicolinearidade
    3. Análise dos resíduos
      1. Normalidade dos resíduos, i.e, se eles variam dentro do intervalo esperado pela dist. normal
      2. Homocedasticidade: Variância de Y é igual, independente de Xi
      3. Verificação de Outliers: se existem valores muito discrepantes do esperado, fazendo uma *tendência no modelo
      4. Independencia dos resíduos


  1. Autocorrelação Espacial
    1. Matriz de vizinhança: comparar a similaridade dos atributos geográficos em feições vizinhas p/ estabelecer o critério de vizinhança wij = 1 se vizinho wij = 0 se não
    2. Índice Global de associação espacial: Índice de Moran I = 1 -> Localzação rlacionada a atributos.
    3. Avaliar se os efeitos de correlação espacial são significativos: Teste de Pseudo Significância
    4. Permutações espaciais aleatórias - distribuição simulada dos ínidices de Moran
        H0: Indice de Moran = 0 (correlação nula)
        H1: Correlação significativa
